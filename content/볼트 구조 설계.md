## 🎯 워크플로우

1. **코드 분석** → `100-Projects/` 폴더에 분석 노트 작성
2. **개념 정리** → `200-Technologies/` 폴더에 기술 노트 작성  
3. **블로그 초안** → `300-Blog-Posts/In-Progress/`에 템플릿 사용
4. **에셋 준비** → `500-Assets/`에 이미지, 다이어그램 저장
5. **발행 후** → `300-Blog-Posts/Published/`로 이동

## 🔧 플러그인 추천

### 필수 플러그인
- **Dataview**: 동적 쿼리와 대시보드
- **Templater**: 고급 템플릿 기능
- **Obsidian Git**: 자동 백업
- **Tag Wrangler**: 태그 관리
- **Kanban**: 칸반 보드로 진행 상황 관리

### 블로그 작성용 플러그인
- **Advanced Tables**: 마크다운 테이블 편집
- **Mermaid**: 다이어그램 작성
- **Code Block Copy**: 코드 복사 기능
- **Image Auto Upload**: 이미지 자동 업로드
